FAQ

How is this different from a standard daily high/low indicator?

Standard indicators measure the entire trading day. This indicator lets you define a custom time window (e.g., 9:30–16:00) and only tracks the high/low inside that window.

Does it work for overnight sessions?

Yes—enable Overnight Session Mode to correctly anchor the window when it crosses midnight. This is essential for instruments like ES or NQ with extended hours.

Will the high and low update in real time?

Yes—the indicator uses OnEachTick calculation, so the high and low lines update continuously as new ticks arrive during the active window.

What does "smart label collision avoidance" do?

It automatically repositions labels to prevent them from overlapping with price bars, keeping your chart clean and readable.

Can I use this on any timeframe?

Yes—it works on any intraday timeframe. The window is time‑based, not bar‑based, so it will correctly track the high/low regardless of your chart's bar interval.

Does it repaint?

The high and low will update as new extremes are reached during the live session. Once the window closes, those values are fixed for that day.

What's the purpose of the window shading?

It provides a visual reference for the active measurement window, making it easier to see exactly when the indicator is tracking the high/low.

Can I display the range in both price and ticks?

Yes—when labels are enabled, the indicator shows both the price range (e.g., 50.00) and the tick distance (e.g., 200 ticks) for quick reference.
